# QRCode_Generator
A simple QRCode generator of C++ with Qt Widget.

## Feature details
1. Generate a bitmap image displayed in QLabel.
2. Support Unicode characters convert to UTF-8 string.
3. Save QRCode bitmap to PNG/JPG/BMP image file.
